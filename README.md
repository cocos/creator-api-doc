# Typedoc JSON Generator for Cocos Creator 3.x API

## Prepare

Install NodeJS v18.

## Install

```sh
$ npm i
```

## Build

```sh
$ npm run build
```

## Unit Test

```sh
$ npm run test
```

## Apply URL Patch

Since the default generated json file doesn't contain the url of source file, run `npm run apply-patch` to enable it.
